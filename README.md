# switch-dashboard
# ระบบกันขโมยตรวจจับการเข้าออก ของตึก Workshop
เป็นการทำงานในรูปแบบ Web Application
อธิบายการทำงาน
มีบอร์ด 2 ตัวคือ Write และ Read
# 1.Write ทำหน้าที่
จะเป็นตัวรับ input ข้อมูลเข้ามาจาก sensor แล้วส่งข้อมูลเข้าไปใน Google sheets โดยจะมี เลข Mac Adress ของเครื่อง ,ช่วงเวลา(DD/MM/YYYY) , ค่าSwitch
# 2.Read ทำหน้าที่
จะเป็นส่งเลข Mac Adress และตัวดึงข้อมูลจาก Google sheets รับข้อมูลมา Output และแสดงสถานะไฟ Revolving light
# Web Application
จะเป็นตัวแสดงผลรับการทำงานของระบบ แสดงค่าตัวเลขการทำงาน และ โซนการทำงานของระบบ








การอธิบายนี้อาจจะช่วยทำให้เข้าใจการทำงานมากขึ้น
